# 📋 TaskFlow - Gerenciador de Tarefas

## 📌 Descrição
O TaskFlow é um sistema simples de gerenciamento de tarefas desenvolvido com HTML, CSS e JavaScript.
Permite organizar tarefas de forma prática com interface baseada em cards.

## ⚙️ Funcionalidades
- Adicionar tarefas
- Definir prioridade (Alta, Média, Baixa)
- Definir prazo
- Marcar como concluída
- Excluir tarefa
- Limite de 10 tarefas em aberto
- Identificar tarefas vencidas
- Armazenamento com LocalStorage

## 🗂️ Estrutura
/projeto
  ├── index.html
  ├── nova-tarefa.html
  ├── em-aberto.html
  ├── concluidas.html
  ├── vencidas.html
  ├── style.css
  └── script.js

## 🧠 Funcionamento
As tarefas são salvas no navegador (LocalStorage) e organizadas automaticamente em:
- Em aberto
- Concluídas
- Vencidas

## 🎨 Interface
- Tema escuro
- Cards organizados
- Cores por prioridade:
  - Alta: vermelho
  - Média: amarelo
  - Baixa: verde

## 🚀 Como executar
1. Baixe os arquivos
2. Coloque todos na mesma pasta
3. Abra o index.html no navegador
